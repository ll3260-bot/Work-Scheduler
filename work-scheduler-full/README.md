# Work Scheduler

A React + Vite + Tailwind CSS dashboard that schedules an 8 AM–6 PM workday, with Pomodoro support, dark mode, and persistent settings via localStorage.

## Features
- Work hours 8:00–18:00 Mon–Fri
- Daily Tasks (10 min), fixed break at 13:00 (30 min), lunch at 15:45 (1 hr)
- 3 project blocks with 30% / 30% / 40% split and daily rotation
- Pomodoro timer with start/pause/reset and persistent settings
- Dark mode and simple settings modal
- Persists data to localStorage

## Run
npm install
npm run dev

Build: npm run build (dist output ready for Netlify)
